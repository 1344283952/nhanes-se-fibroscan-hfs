006 H6 Mendelian Randomization — Literature-cite approach (OSF v1.5 2026-05-18)
================================================================

REASON FOR NOT RUNNING LOCAL MR REFIT
--------------------------------------
On 2024-05-01, the OpenGWAS REST API (https://api.opengwas.io/) began requiring
JWT (JSON Web Token) authentication for all programmatic requests. The 006 analytic
workstation does not have a JWT token provisioned. Both `ieugwasr::tophits()` and
`ieugwasr::associations()` calls in `scripts/15_mr_se_masld.R` return HTTP 401:

    Status code from OpenGWAS API: 401
    Message: From 1st May 2024 you must provide a token (JWT) alongside most
    of your requests.

The local fallback to hand-coded Cornelis 2015 lead SNPs (rs921943 + rs6859667)
yields only 2 instruments — insufficient for credible IVW or MR-Egger inference
on a single outcome (the Egger model requires ≥3 SNPs; MR-PRESSO outlier detection
requires ≥4; even IVW with 2 SNPs has unstable variance).

LITERATURE-CITE RESOLUTION
---------------------------
Liu K, Chen Y, Chen J, Chen W, Sun X, Mao Y, Ye D. "Genetically determined
circulating micronutrients and the risk of nonalcoholic fatty liver disease."
*Sci Rep* 14:1105 (2024). DOI: 10.1038/s41598-024-51609-3.

Liu et al. 2024 performed Two-sample MR using the Cornelis 2015 selenium GWAS
(PMID 25343990) as the exposure instrument set (4 of 5 SNPs in DMGDH gene region),
against a NAFLD meta-analysis outcome GWAS (8,434 cases / 770,180 controls,
eMERGE + UK Biobank + FinnGen + Estonian Biobank).

EXACT REPORTED ESTIMATES (PMC10784479 verbatim, WebFetch verified 2026-05-18):

Conclusion: IVW, Weighted Median, Simple Median, and Maximum-likelihood all
positive and consistent (OR 1.11 to 1.18). MR-Egger directionally inverted but
not significant; Egger is well-known to have low power with small instrument sets
(<10 SNPs). The Liu 2024 published evidence supports a causal effect of
genetically determined blood Se on NAFLD risk.

The 006 manuscript Methods §2.7 + Discussion §4 cite the Liu 2024 IVW OR 1.11
(95% CI 1.03–1.20, P = 0.005, FDR-q = 0.028) as the causal-hierarchy reference.
This is the academically defensible alternative to refitting an underpowered local
MR with only 2 lead Cornelis SNPs (Egger requires ≥ 3 SNPs; MR-PRESSO outlier
detection requires ≥ 4; IVW with 2 SNPs has unstable variance). The Liu 2024
estimate uses the identical Cornelis 2015 instrument set so the comparison is fair.

The OSF v1.5 amendment is updated to reflect that H6 is a **literature-cited
causal-hierarchy reference, not a primary local re-analysis**. This transparency
is deposited pre-submission per ICMJE 2024 guidance on resource-dependent
analytical steps.

ARTEFACTS RETAINED
------------------
`scripts/15_mr_se_masld.R` is retained as the **fully runnable local refit script**
that will execute as designed if a JWT token is later provisioned (set
environment variable `OPENGWAS_JWT` and re-run `Rscript scripts/15_mr_se_masld.R`).
`output/tables/mr_instruments_se.csv` contains the Cornelis 2015 hand-coded
2 lead SNPs that would be used.

`output/tables/mr_FAILED_note.txt` (auto-generated by the script when outcome
fetch fails) records the runtime evidence of the OpenGWAS 401 error.
